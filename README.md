# Fitness Calculator

An android application for essential fitness calculations.

[Play Store](https://play.google.com/store/apps/details?id=com.hanson.android.fcalc)

## Features

1. One Rep Max Calculator
1. TDEE & BMI Calculator
1. Wilks Coefficient
